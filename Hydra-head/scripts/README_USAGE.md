Usage examples
--------------
- Generate keys: ./scripts/generate_keys.sh
- Start docker compose: docker compose up --build
- Init head: ./scripts/init_head_example.sh
- Commit blueprint (example): node scripts/commit_blueprint_example.js
- Submit new tx: node scripts/newtx_example.js
- Orchestrator API: POST http://localhost:9000/event { "event": { ... } }
